## Encrypt/Decrypt files

```
`//for binary file, output > gpgtest.txt.gpg
gpg -r pengjianqing@gmail.com -e gpgtest.txt
//for text, output >  gpgtest.txt.asc
gpg -r pengjianqing@gmail.com -ae gpgtest.txt
gpg -d gpgtest.txt.gpg`
```
