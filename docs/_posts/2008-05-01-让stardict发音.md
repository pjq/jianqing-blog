装好stardict后，再装上真人语音库，
语音库不一定非要放到/usr/share下，只要更改首选项里的音效路径就可以了
比如我的：
/media/sda11/usr/share/WyabdcRealPeopleTTS/
或/media/sda7/linux/ubuntusoft/WyabdcRealPeopleTTS/
应该就差不多了，
但我的好像哪里出了点问题，在网上找了一下，是stardict需要esd
然后
ls /etc/init.d/
有一个esound，然后启动它，
/etc/init.d/esound start
再重启stardict
发音正常
再将它添加到启动项里
rc-update add esound default
