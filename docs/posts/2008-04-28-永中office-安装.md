1.先下载安装文件

EIO2007_Trial_ZH_Lin.tar.gz

2.解压：sudo tar xvzf EIO2007*

3.增加可执行权限：

cd 到安装目录

sudo chmod +x setup*

4.安装：

su

./setup

5.安装完毕

6.安装字体：

sudo mkdir -p /usr/share/fonts/zh_CN/TrueType

sudo ln -s /media/sda1/WINDOWS/Fonts/*.ttf /usr/share/fonts/zh_CN/TrueType/          #直接链接到XP下的字体

sudo ln -s /media/sda1/WINDOWS/Fonts/*.ttc /usr/share/fonts/zh_CN/TrueType/
sudo ln -s /media/sda1/WINDOWS/Fonts/*.TTF /usr/share/fonts/zh_CN/TrueType/            # 这里好像包含了很多中文字体
sudo fc-cache
7.至此安装完毕，启动eio
